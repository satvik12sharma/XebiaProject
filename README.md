# Enterprise Workforce Management Platform with AI Operations Assistant

A centralized, enterprise-grade web platform designed to manage the entire employee lifecycle with an integrated role-aware AI Operations Assistant.

## Project Structure
- `/backend`: Node.js + Express REST API server, authentication, role verification, database integrations.
- `/frontend`: Vite + React single-page application with modern responsive layout, analytics dashboards, and interactive AI chatbot.

## Core Modules
1. **Authentication & User Management**: Secure JWT logins, RBAC, password security, account lockouts.
2. **Organization & Employee Management**: Department hierarchies, digital profiles, and lifecycle tracking.
3. **Recruitment Management**: Hiring stages, pipeline tracking, and resume analytics.
4. **Attendance Management**: Daily logging with GPS verification, QR code support, and overtime checks.
5. **Leave Management**: Balance allowances, application forms, and approval flows.
6. **Payroll Management**: Auto-calculations, bonus integration, and payslip generator.
7. **Project & Task Management**: Sprints, Kanban board, priority settings, and review loops.
8. **Performance, Asset, & Help Desk Modules**: Reviews, asset tracking, IT support tickets.
9. **AI Operations Assistant**: Persistent HR policy assistant, resume analyzer, and performance advisor.
